Python local olarak proje çalıştırmak.

1- Python sitesinden python indir, kurulum yap.
2- pip install virtualenv    -> sanal ortam için paket kurulumu yap.
3- mkdir project-name        -> proje dosyası oluştur.
4- cd project-name           -> proje dizinine gir.
5- python -m venv venv       -> proje dizininde sanal ortam kur.
6- touch requirements.text   -> kullanılacak paketlerin olduğu bir txt dosyası hazırla kaydet
7- venv\Scripts\activate.bat -> sanal ortamı aktif hale getir. consolde (venv) yazacak.
8- python -m pip install -r requirements.txt -> sanal ortama bağımlılık paketlerini yükle.
9- Daha sonra IDE'nin ayarlarından project structure venv olarak ayarlanacak.
10- Ve başlamaya hazırız. Lokalimizdeki paketlerden tamamen soyutlanmış 
bir sanal ortamda proje geliştirebiliriz.


Paketlerimizin yüklenip yüklenmediğini kotnrol etmek için:
1- pip list -> (venv) içerisinde

Paket yükleme ve kaldırma yönetimi:

Yöntem 1:
1- pip freeze > requirements.txt      -> Bize sanal ortamda kurulu paketleri getirir.
2- pip uninstall -r requirements.txt {-y}  -> Paket kaldırmak için kullanılır (-y hepsini birden)

Yöntem 2:
1- pip uninstall -r {-y} <(pip freeze) -> Tek komutta silmek
